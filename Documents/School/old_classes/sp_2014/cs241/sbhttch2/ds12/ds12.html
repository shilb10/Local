Discussion Section for December 5, 2013: Very Simple Web Server


In Discussion Section

In ds12, you will write a networking program that will listen on a port number specified in the command line, print out the received content and echo back an HTTP text/html response containing “Hello CS241!!!\n” back to the client. The program only need to support one connection at a time.

Compiling and Running

To compile and run, use the following commands from a terminal on a Linux machine:

$ make

$ ./ds10 port_number

To send a command to your program, you could access the url 127.00.1:port_number from your web browser and that will send an HTTP packet to your program.
